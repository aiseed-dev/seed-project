# 利用規約(草案)

**専門家レビュー前提のたたき台**(docs/08 Phase 8)。

## 1. サービスの性格
- 本サービスは固定種・在来種の種苗等を利用者間で交換・販売・譲渡するための
  「場」を提供するものであり、運営者は取引の当事者にならない。
- 決済機能は提供しない。金銭の支払いは当事者間で行う(後払いを原則とし、
  方法は当事者の協議による)。

## 2. 種苗法の遵守(最重要)
- **品種登録された品種(登録品種)の種苗を無断で譲渡・販売・出品することを
  禁止する。** 出品時に「品種登録されていない一般品種である」ことの確認を
  必須とし、違反が確認された出品は予告なく停止・削除する。
- 種苗業者(反復継続して販売する者)は、種苗法22条の指定種苗の表示
  (氏名・住所・生産地、種子は発芽率等)を出品に含めなければならない。
  表示内容の正確性は出品者の責任とし、運営者は表示欄を提供するに留まる。

## 3. 個人の家庭採種品の性質表示
- 個人出品の種苗は原則「家庭採種・発芽保証なし」の性質表示を伴う。
  これは品質保証の否認であり、表示義務の回避手段ではない。
  発芽・生育・純度は保証されない。

## 4. 送料
- **販売・譲渡は送料別**とし、実費を当事者間のメッセージで合意する。
- **交換は各自が自分の発送分を負担**する(送料の注記なし)。

## 5. 生産物(生鮮野菜)
- 郵送する生産物には食品表示(名称・原産地・生産者・収穫日・保存方法)を
  必須とする。郵送で業として販売する者は特定商取引法に基づく表示を行う。
- 加工品・販売に許可を要する食品は取り扱わない。

## 6. 個人間取引の免責
- 取引・受け渡し・支払いに関する紛争は当事者間で解決する。
  運営者は通報に基づく事後審査(出品停止・アカウント停止等)を行うが、
  取引の履行・品質について責任を負わない。

## 7. 辞典コンテンツ
- 辞典への投稿(記事・写真)は **CC BY-SA 4.0** で公開されることに
  同意したものとする。

## 8. 禁止事項・その他
- 登録品種の出品、虚偽表示、スパム、なりすまし、その他法令違反。
- アカウント停止・出品削除の基準と手続、規約の変更手続は本規約に定める。
