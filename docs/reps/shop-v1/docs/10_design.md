# 10 デザイン指針(種苗カタログ様式)

参照イメージ: 野口のタネのカテゴリページ(生成り背景・整然としたカード・
緑のキャッチコピー・橙の価格とボタン)。メルカリ風のフリマUIではなく、
**種苗カタログの様式**を全アプリ共通の視覚言語とする。
情報密度が高く、セールス先(種苗店)が日常的に見ている画面様式でもある。

## 配色(core/constants.dart に定義)

| 用途 | 色 | 例 |
|---|---|---|
| 背景(生成り) | #F5F1E6 | 画面背景 |
| 面(カード) | #FFFFFF | カード・シート |
| 文字(主) | #33302A | 見出し・本文 |
| 緑(基調) | #5A7A2E | キャッチコピー・チップ枠/文字・「交換」バッジ |
| 橙(強調) | #C0761A | 価格・主ボタン(カートに入れる/申込みを送る) |
| 青(補助) | #3A6B8A | 「譲渡」バッジ |
| 無効 | #A9A297 | 取引中・品切れ状態のボタン |

主ボタンは橙のみ(1画面に1種類の強調)。Material 3 の ColorScheme に
上記をマッピングし、個別Widgetでの直接指定は禁止。

## 出品カードの解剖(core の ListingCard。全アプリ共通)

上から順に:

1. **写真**(白背景・正方形寄り)。左上に種別バッジ
   (販売=橙 / 交換=緑 / 譲渡=青)
2. **キャッチコピー**(緑・小)… shared.varieties.summary から自動表示。
   辞典が育つほどカードが充実する(未紐付け品種は非表示)
3. **品種名**(太字)
4. **チップ**(白地・緑枠・緑字・角丸小): 固定種 / 在来種 / 自家採種 /
   ✓店舗名(認証店)
5. **スペック行**(ラベル: 値 の2列、12px):
   - 蒔き時 … **辞典(品種マスタの栽培情報)から自動表示**。出品者は入力しない
   - 生産地/採種地(指定種苗表示 or 個人の地域)
   - 内容量・数量表記
   - 発芽率・種子消毒(requires_seed_label 時。参照イメージの
     「種子消毒あり」バッジ・「採種地: 長野県」に対応)
   - requires_seed_label 時はカード内に「指定種苗表示」の小見出しを付け、
     詳細は出品詳細で全事項(表示者氏名住所含む)を表示

**業者品と個人品の対照(同じスペック行の同じ位置で見せる)**:
   - 業者品(requires_seed_label=true): 発芽率あり(例「発芽率 80%以上」)。
     正規の指定種苗表示
   - 個人品(no_warranty=true): 発芽率の位置に「家庭採種・発芽保証なし」の
     控えめな注記(グレー小)。発芽率は表示しない
   この有無が買う人にとっての見分け線になる。バッジで煽らず、
   スペック行の一項目として淡々と対照させる
6. **条件行**: 価格(橙・税込表記)+「送料別」(小・グレー)/
   「希望: ○○」(交換。送料は各自負担のため注記なし)/
   「無償でお譲りします(送料別)」(譲渡)。
   受け渡し(直接/郵送)と支払い既定(後払い等)を小さく添える。
   生産物の郵送は「食品表示」、郵送の事業者は「特商法表示」の
   小見出しをカードに付け、詳細で全事項を表示
7. **数量+主ボタン**(橙・全幅): 数量ステッパー(販売品のみ)+
   「カートに入れる」(全種別共通)。
   取引中・終了はグレー「ただいま取引中です」「終了しました」。
   店側在庫APIで在庫なしの店舗出品はグレー「ただいま品切れ中です」
   (参照イメージと同じ位置・様式)。在庫の表示粒度は後日決定
   (テスト段階は正確な数を小さく添える)

## 画面への適用

- 品目ページ(統合アプリ CropHubScreen): 参照イメージと同じ構成。
  品目名見出し+パンくず(ホーム / 分類 / 品目)、件数表示、
  並び替え、カードグリッド(モバイル2列 / Web 3〜4列)
- 交換用アプリの分類別一覧・検索結果も同じ ListingCard を使用
  (画面構成はメルカリ的な回遊でも、カードはカタログ様式で統一)
- 販売用アプリの出品テーブルのモバイル表示にも同カードの縮小版
- 出品詳細: カードの拡大形。写真スワイプ→キャッチコピー→品種名→
  チップ→スペック表→条件→出品者→ボタン、の同じ順序を保つ

## 文言

- ボタンは動詞から。「カートに入れる」「申込みを送る」「出品する」「編集を提案」
- 感嘆符・過剰な絵文字を使わない。種苗店の商品ページの落ち着いたトーン
- 状態表示は参照イメージに倣い丁寧語(「ただいま取引中です」)
