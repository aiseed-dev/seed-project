# プライバシーポリシー(草案)

**専門家レビュー前提のたたき台**(docs/08 Phase 8)。

## 取得する情報
- アカウント情報: メールアドレス・パスワード(PocketBase が保持。
  業務データベースには保存しない)
- プロフィール: 表示名・地域(都道府県)・自己紹介(任意)
- 取引情報: 出品・カート・申込み・メッセージ・評価(業務データベース)
- 指定種苗表示として出品者が自ら記載した氏名・住所(公開される)

## 利用目的
- サービスの提供(取引の仲介・通知メールの送信・辞典の運営)
- 種苗法・特定商取引法等への対応(表示義務・通報対応)
- 不正利用の防止(通報に基づく事後審査)

## 第三者提供
- 行わない。ただし法令に基づく開示請求には応じることがある。
- メッセージの内容は取引の相手方にのみ表示される。

## 保管と削除
- サーバーは国内で運営し、バックアップは暗号化された外部ストレージに置く。
- アカウント削除の申請により、法令上の保存義務があるもの(経理記録等)を
  除き削除する。

## 問い合わせ先
- 運営者の連絡先(メールアドレス)を公開ページに記載する。
