# 野菜辞典への協力ガイド(自然農法の実践者のみなさまへ)

伝統野菜辞典は、固定種・在来種を育てる人たちの実践知を集める共同の辞典です。
とくに**自然農法・有機栽培での育て方のコツ**を歓迎します。記事には
「自然農法」の専用セクションがあります(歴史 / 栽培方法 / **自然農法** /
採種方法 / 料理 / 出典)。

## 協力のしかた(3通り)

### 1. アプリから書く(いちばん簡単・GitHub 不要)

1. 種の交換アプリでアカウントを作る(メール確認あり)
2. 品種のページを開き「**編集を提案**」を押す
3. 「自然農法」欄に実践のコツを書く(例: 不耕起での播種のタイミング、
   無肥料での生育の見立て、その土地での相性、失敗談も歓迎)
4. 「提案を送る」→ 編集者の確認ののち公開されます。却下される場合も
   理由がメールで届き、直して再提案できます

書いた内容は **CC BY-SA 4.0** で公開されます(みんなの知見が
囲い込まれないための約束です)。

### 2. 原稿ファイルで協力する(GitHub が使える方)

正本は `vegitage-data/web/<地域>/` の Markdown です。

- 概要: `<作物>.md` / 歴史: `history/<作物>.md` /
  栽培: `cultivation/<作物>.md` / 料理: `cuisine/<作物>.md`
- 栽培原稿の中の「自然栽培・有機栽培のポイント」の節(見出しに
  「自然」または「有機」を含む H2/H3)は、辞典取込時に自動で
  「自然農法」セクションへ入ります
- プルリクエストを送ってください。取込は運営が
  `backend/scripts/vegimport.py --refresh` で行います

### 3. 編集者(editor)として協力する

提案の確認・承認を手伝ってくださる方には editor 権限をお渡しします
(運営が管理アプリで付与)。アプリのマイページに「編集提案の承認」が
現れ、スマホから差分を見て承認・却下できます。

## 書くときの目安

- その土地(地域・気候)を明記する。「埼玉県の平地では…」
- 何年やっての実感かを添える。「3年続けての傾向として…」
- 断定より観察を。うまくいかなかった記録にも価値があります
- 出典があれば「出典」欄へ(書籍・種苗店カタログ・聞き書き など)
